Observações:

  • É recomendado enviar no máximo 200 mensagens por conta por dia
  • É recomendado usar uma janela do Chrome exclusivamente para o WhatsApp Web e o Disparador, dessa forma você poderá usar e operar outras guias do Chrome
  • Se os destinatários forem inseridos manualmente (digitados), pressione a tecla Enter para inserir uma quebra de linha para cada número
  • Insira os números sem caracteres especiais nesse formato: DDD+Número
  • O tempo de envio padrão para cada mensagem é aleatório de 5 há 20 segundos, isso ajuda minimizar o bloqueio da conta
